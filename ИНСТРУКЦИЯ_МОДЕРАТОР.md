# Инструкция для модератора

## Вход в режим модератора

**Команда:** `/moderator`

**Пароль:** `SimpPhysXRedactor123`

## Как использовать режим модератора

1. Отправьте команду `/moderator` в бот
2. Введите пароль `SimpPhysXRedactor123`
3. Выберите раздел (Комиксы, Мемы, Исторические справки, Физика в фильмах, Календарик, Конспекты)
4. Выберите действие:
   - **➕ Добавить** - добавить новую запись
   - **🗑️ Удалить** - удалить запись по ID
   - **👁️ Посмотреть все** - посмотреть все записи с их ID
   - **✏️ Редактировать** - отредактировать запись по ID

## Процесс добавления

1. Нажмите "➕ Добавить"
2. Отправьте **фото**
3. Отправьте **текст** для записи

## Процесс редактирования

1. Нажмите "✏️ Редактировать"
2. Введите **ID** записи (можно посмотреть через "👁️ Посмотреть все")
3. Отправьте **новое фото** (или "пропустить" чтобы оставить старое)
4. Отправьте **новый текст**

## Процесс удаления

1. Нажмите "🗑️ Удалить"
2. Введите **ID** записи для удаления

## Выход из режима модератора

Нажмите кнопку "🚪 Выйти из режима модератора" в меню модератора.
