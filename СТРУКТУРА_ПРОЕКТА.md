## 🖼️ Как работать с локальными файлами

### Вариант 1: Использование через базу данных (рекомендуется)

1. **Добавление через режим модератора:**
   - Запустите бота
   - Отправьте `/moderator`
   - Введите пароль: `SimpPhysXRedactor123`
   - Выберите раздел → Добавить
   - Введите название
   - Отправьте фото (или пропустите)
   - Отправьте текст (или пропустите)

2. **Фото сохраняется в Telegram:**
   - Telegram хранит файлы на своих серверах
   - В базе данных сохраняется только `file_id` (идентификатор файла)
   - `file_id` используется для получения файла из Telegram

## 📝 Формат данных в базе данных

### Таблица `comics`
- `id` - уникальный идентификатор
- `title` - название комикса
- `content` - описание/текст (может быть NULL)
- `file_id` - идентификатор фото в Telegram (может быть NULL)

### Таблица `memes`
- `id` - уникальный идентификатор
- `title` - название мема
- `content` - описание/текст (может быть NULL)
- `file_id` - идентификатор фото в Telegram (может быть NULL)

### Таблица `history`
- `id` - уникальный идентификатор
- `title` - название справки
- `content` - текст справки
- `file_id` - идентификатор фото в Telegram (может быть NULL)

### Таблица `movies`
- `id` - уникальный идентификатор

## 🔧 Как добавить контент вручную в код (для демонстрации)

Если нужно быстро добавить контент для демонстрации, можно использовать локальные файлы:

1. Поместите файлы в папку `media/` в соответствующие подпапки
2. Используйте функцию загрузки файлов в Telegram
3. Получите `file_id` и добавьте в базу данных

**Пример кода для добавления:**
```python
from bot import init_db, add_comic
from telegram import Bot
import asyncio

async def upload_and_add():
    bot = Bot("YOUR_BOT_TOKEN")
    init_db()
    
    # Загрузите файл
    with open("media/comics/comic1.png", "rb") as photo:
        message = await bot.send_photo(
            chat_id=YOUR_CHAT_ID,
            photo=photo,
            caption="Тестовый комикс"
        )
    
    # Получите file_id
    file_id = message.photo[-1].file_id
    
    # Добавьте в базу данных
    add_comic("Название комикса", file_id, "Описание комикса")
    print(f"Добавлено! file_id: {file_id}")

# Запустите
asyncio.run(upload_and_add())
```

## ⚠️ Важные замечания

1. **file_id не является постоянным:**
   - `file_id` может измениться при перезапуске бота
   - Для постоянного хранения используйте `file_unique_id` (но он не работает для отправки)

2. **Локальные файлы:**
   - Локальные файлы не загружаются автоматически
   - Нужно вручную загрузить их в Telegram и получить `file_id`

3. **Рекомендация:**
   - Используйте режим модератора для добавления контента
   - Это самый простой и надежный способ
